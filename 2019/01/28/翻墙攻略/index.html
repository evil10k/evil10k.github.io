翻墙攻略

一、利用vultr购买搭建服务器
1.注册
2.选择服务器及其地址
（我的选择如下）
Server Location： Tokyo
Server Type: CentOS7 x64
Server Size: 20 GB SSD
Additional Features: Enable IPv6
Server Hostnam……
这些都选完了之后点Deploy Now
3.绑定银行卡
（成功后会获得自己的服务器信息~）
/到此，预备工作就基本完成了。接下来就要将刚才购买的服务器连接起来即可。/

二、连接服务器
1.下载Xsehll
2.连接linux
（1）点击文件、新建
（2）将服务器IP填写在“主机（H）”后
（3）输入用户名（并选择记住用户名）
（4）输入密码（并选择记住密码）

三、使Mac OS 连接VPS
打开Terminal，输入相关信息（密码不会显示出来）

四、搭建shaodowsock
1.下载ss（复制“git clone https://github.com/Flyzy2005/ss-fly”）
2.输入“ss-fly/ss-fly.sh -i 要设置的密码”
